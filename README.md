# Asafo Print POS - Passport.js Version 🔐

User login with Passport.js + Express, easy to deploy on Hostinger!

## 📦 Features
- ✅ User Signup & Login with Passport.js Local Strategy
- ✅ User-specific Materials saved in data.json
- ✅ Price Calculator
- ✅ Beautiful Dark Theme
- ✅ Easy to Deploy on Hostinger

## 🚀 Local Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the app:
   ```bash
   npm start
   ```
3. Open http://localhost:3000

## 🌍 Deploy to Hostinger

### Step 1: Upload Files
1. Zip the entire folder (except node_modules and data.json)
2. Log into Hostinger
3. Go to Websites → Your Domain → File Manager
4. Upload the zip to public_html → Extract it

### Step 2: Install via SSH (Recommended)
1. Log into Hostinger SSH
2. cd to your app folder
3. npm install
4. npm start (use PM2 to keep it running)
